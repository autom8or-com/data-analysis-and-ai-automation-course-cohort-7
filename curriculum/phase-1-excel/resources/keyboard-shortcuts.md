# Excel Keyboard Shortcuts
## Phase 1 — PORA Academy Cohort 7

> To be completed.

## Navigation
| Shortcut | Action |
|---|---|
| `Ctrl + End` | Jump to last used cell |
| `Ctrl + Home` | Jump to A1 |
| `Ctrl + Arrow` | Jump to edge of data region |
| `Ctrl + Shift + End` | Select to last used cell |

## Data
| Shortcut | Action |
|---|---|
| `Ctrl + T` | Create Table |
| `Ctrl + Shift + L` | Toggle AutoFilter |
| `Alt + =` | AutoSum |
| `Ctrl + D` | Fill down |

## Workbook
| Shortcut | Action |
|---|---|
| `Ctrl + Z` | Undo |
| `Ctrl + S` | Save |
| `F2` | Edit active cell |
| `F4` | Repeat last action / toggle absolute reference |
